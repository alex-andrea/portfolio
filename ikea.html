Website Redesign / August 2018

I worked in a group of three to evaluate and redesign the IKEA website to provide a better user experience. We spent one week first evaluating and then redesigning the website using all IKEA images to present a more polished version of the IKEA brand. We spent an additional week conducting user testing via UsabilityHub.

We used UsabilityHub to test our wireframe against a wireframe of IKEA's current layout. We gave users instructions to find a specific IKEA product. Our layout resulted in a 1779.43% increase in task completion compared to IKEA's current layout.
